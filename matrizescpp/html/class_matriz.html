Ele realiza alocacao dinamica de um array bidimensional que armazena a matriz a ser processada

Parameters
_nleh a quantidade de linhas que a matriz terá
_ncé a quantidade de colunas
  1. Se _nl ou _nc iguais a zero a matriz terá todas as dimensoes definidas como zero
  2. Se as dimensoes forem negativas, o programa será assassinado

operator = copia o estado da matriz fornecida para o objeto atual
